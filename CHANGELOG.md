# Changelog

All notable changes to the GateKey Android Client are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.4.2] - 2026-01-11

### Changed
- Version bump to align with server release
- Minor stability improvements

## [1.4.1] - 2026-01-10

### Added
- IPv6 support for VPN connections
- Dual-stack networking capability

### Changed
- Updated OpenVPN submodule with IPv6 changes

### Security
- Security feature enhancements

## [1.3.2] - 2026-01-09

### Changed
- Updated login flow for SAML compatibility
- Improved SSO authentication handling

## [1.3.1] - 2026-01-07

### Added
- WireGuard mesh network compatibility
- Support for mesh VPN topologies

## [1.3.0] - 2026-01-06

### Added
- WireGuard VPN protocol support
- WireGuard tunnel configuration and management

### Changed
- Updated to Java 21
- Modernized build configuration

### Fixed
- WireGuard connection stability issues

## [1.0.0] - 2026-01-05

### Added
- Initial release of GateKey Android Client
- OpenVPN protocol support
- Certificate-based authentication
- OIDC/SSO login integration
- Gateway selection and connection management
- VPN status monitoring
- Split tunneling support
- Background VPN service
- Material Design 3 UI
- Dark mode support
